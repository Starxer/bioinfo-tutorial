# 生信入门小组课程大纲

## 第一阶段：基础工具与原理

1. **认识 Linux 与命令行**
   - 文件系统、常用命令（ls/cd/mv/cp/find/grep）
   - 权限、重定向、管道
   - 理解"一切皆文件"

2. **测序技术简介**
   - Sanger → NGS → TGS 的演进
   - NGS 测序原理（文库构建、桥式PCR、边合成边测序）
   - 从 .fastq 到 .bam 再到 .vcf：常见文件格式及含义

3. **环境管理：conda / mamba**
   - 创建、激活、删除环境
   - 用 conda 装生信软件 vs pip 装 Python 包——何时用哪个
   - 写一个 environment.yml

4. **公共数据库与序列比对**
   - NCBI / Ensembl / UCSC Genome Browser 的用法
   - BLAST 在线搜索 vs 本地比对
   - 序列比对基本思想（精确匹配 vs 近似匹配，bwa/minimap2）

5. **VS Code 与开发效率**
   - 代码编辑、远程连接（Remote-SSH）、终端集成
   - 简单的 git 工作流：init → add → commit → push
   - 项目目录组织规范（data/ scripts/ results/ docs/）

## 第二阶段：实战分析

6. **用 Python 和 R 画图**
   - 火山图（差异表达可视化）
   - 进化树（系统发育基础）
   - matplotlib / seaborn / ggplot2 基本用法

7. **完整运行一次转录组分析流程**
   - 从原始 .fastq 到差异表达基因列表
   - 理解每一步在做什么（质控 → 比对 → 定量 → 差异分析）
   - 不把工具当黑箱
