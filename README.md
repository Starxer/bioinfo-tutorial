## 使用方式

本教程为 Obsidian 笔记格式。克隆后在 Obsidian 中打开文件夹即可获得完整的双向链接体验。

```bash
git clone https://github.com/Starxer/bioinfo-tutorial.git
```

所有 `.md` 文件也可直接用 VS Code、Typora 等任意 Markdown 编辑器打开阅读。
